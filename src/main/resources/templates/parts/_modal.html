Json Format

table-> Конвертирование json объекта в data table

<-json Конвертирование Data table в json

Пример json: { "путь до параметра" : "значение" }

Пример data table: | путь до параметра | значение |

Xml Format

table-> Конвертирование xml объекта в data table

table cut-> Конвертирование xml объекта в data table (без name-space в названии тегов и атрибутов)

<-xml Конвертирование Data table в xml

Пример xml: <root><node>значение</node></root>

Пример data table: | root.node | значение |

Query Format

create all->Получить шаг создания таблицы из query.sql(Все поля)

create N/N->Получить шаг создания таблицы из query.sql(только поля not null)

check all->Получить шаг проверки таблицы из query.sql(Все поля)

check N/N->Получить шаг проверки таблицы из query.sql(только поля not null)

поля not null обозначены знаком *
